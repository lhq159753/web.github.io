这是一篇liunx学习手册的心得🦁


这仅仅只是一个不如渗透的一个小开始

🐧ls 查看 ls -l查看权限 lspci：查看网卡信息 lspci -vvv ：查看网卡驱动

🐧 touch 创建文件 ethtool -i eth0： 查看网卡驱动版本

🐧 mkdir 创建目录 mkdir -p

🐧 cd 进入文件或目录

🐧 rm 删除

🐧 cat 查看文件适合文件较少 cat -n显示行号

🐧 more 查看文件适合较多 分页浏览

🐧 less 查看文件可以搜索目标

🐧 head -n 查看目标文件所需要的几行

🐧 mv 移动文件夹

🐧 cp 复制文件 cp -p

🐧 ln 生成链接文件 ln -s生成为软链接 类似于windows快捷方式

🐧 useradd name 创建用户 userdel删除用户信息

🐧 groupadd 创建组 groupdel删除

🐧 权限更改 chmod ugoa +-= rwx 文件或目录（touch mkdir）

🐧 r：读权限，用户可以读取文档的内容，如用cat，more查看

🐧 w：写权限，用户可以编辑文档

🐧 x：该目录具有可以被系统执行的权限

🐧 x= 1 w= 2 r=4

🐧 file（文件） r:cat/more/head/tail/less

                w:vim

                 x:脚本 command

🐧 directory（目录） r:ls

                           w:touch/mkdir/rmdir/rm



chown：改变文件所属用户  chown user file

chgrp ：改变文件或目录所属组

umask -S查看权限更改默认权限
